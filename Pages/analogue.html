HexChem

  • Genre: Strategy
  • Status: Completed
  • Date: 2025

    • This strategy game was created by myself and another team member for the Campus Game Jam in 2025. The theme for this jam was "Strange Side Effects". The jam took place over the course of 48 hours, in which we conceptualized, designed and created the game. In HexChem, players race to create the all-curing WonderDrug, by collecting and comining diferent Hex Molecules to create medicinal chemicals which cause various side effects. HexChem won the award for most addictive game, and also placed first in the board game category.

    MadLab

  • Genre: Roll & Write
  • Status: Completed
  • Date: 2024

    • This roll & write game was made for an Assignment in my Game Design course. We had to create a game in the Roll & Write genre, which is a type of game where players roll dice and then write or mark the results on a score sheet. Players assume the role of mad scientists in a race to create the most efficient robot. MadLab is a fun and engaging game that combines elements of luck and strategy in a unique way.
